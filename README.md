# 🏢 Employee Management System (Flask Backend)

A production-ready Employee Management System built using Flask and PostgreSQL.  
This backend application provides REST APIs to manage employees with image upload support, middleware logging, and global error handling.

---

## 🚀 Features

- ✅ Add Employee (with Image Upload)
- ✅ Get All Employees
- ✅ Delete Employee
- ✅ PostgreSQL Database Integration
- ✅ RESTful API Architecture
- ✅ Request Logging Middleware
- ✅ Global Error Handling
- ✅ Structured Project Architecture

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|----------|
| Python | Programming Language |
| Flask | Web Framework |
| Flask-SQLAlchemy | ORM |
| PostgreSQL | Database |
| Werkzeug | Secure File Upload |
| Logging | Request Tracking |
